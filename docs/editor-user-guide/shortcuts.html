Keyboard Shortcuts

快捷键和鼠标操作

这页整理当前编辑器的工具切换、Scene View 导航、选择、文档操作、层级菜单和面板操作。第一次上手时，先记住最小集合，再按分类查细节。

先看最常用 返回用户指南

最小记忆集合

完成第一个可保存修改时，优先记住下面几组操作。

工具

Q 选择，W 移动，E 旋转，R 缩放。

视图

F 聚焦选中对象，Esc 取消活动操作，滚轮推进/拉远。

文档

Cmd/Ctrl + S 保存，Cmd/Ctrl + Z 撤销，Delete 删除选中对象。

工具与 Transform

工具快捷键只切换当前编辑工具，不会直接修改场景数据。真正的 Transform 变化来自 gizmo 拖拽或 Inspector 数值编辑。

按键或入口 作用 使用场景
Q切换到选择工具。普通选择、框选前的基础模式。
W切换到移动工具。显示移动 gizmo，可沿轴向、平面或视图平面移动。
E切换到旋转工具。显示旋转 gizmo，绕 X/Y/Z 单轴旋转。
R切换到缩放工具。显示缩放 gizmo，可单轴缩放或等比缩放。
世界 / 本地在 Scene View 工具条切换 Transform 坐标空间。需要按世界方向或对象自身方向移动、旋转、缩放时使用。
移动手柄轴向箭头、平面方块和中心自由手柄同时可用。精确移动时用轴向或平面；快速拖动时用中心自由手柄。
吸附在 Scene View 工具条切换移动、旋转、缩放步进吸附。需要按固定网格、角度或比例编辑时使用。
放置在 Scene View 工具条切换关闭、XZ 地面或场景表面放置。把对象放到地面或已有表面上时使用。
对齐 / 分布多选后从 Scene View 工具条执行位置对齐或分布。需要快速整理一组选中对象的位置时使用。

Scene View 导航

Scene View 相机操作只改变观察视角，不写入 authoring source。飞行视角中，滚轮用于调整飞行速度。

按键或鼠标 作用 说明
F聚焦当前选中的 GameObject。需要先选中一个可聚焦对象。
Esc取消当前 gizmo 拖拽或活动操作。不等同于撤销历史。
滚轮推进/拉远摄像机。右键飞行模式下改为调整飞行速度。
中键拖拽平移 Scene View。适合微调视图构图。
Alt + 左键拖拽环绕 Scene View。围绕当前视图目标旋转观察。
Alt + 中键拖拽平移 Scene View。和中键拖拽类似。
Alt + 右键拖拽推进/拉远 Scene View。适合没有滚轮或需要连续 dolly 时使用。
右键拖拽进入飞行视角并转动相机。按住右键时可使用 WASDQE 移动。
右键 + W/S飞行视角前进 / 后退。只移动相机，不移动对象。
右键 + A/D飞行视角左移 / 右移。按相机朝向的左右方向移动。
右键 + Q/E飞行视角下降 / 上升。用于快速调整观察高度。

鼠标操作

操作 作用 说明
左键选择对象。在空白区域拖拽可以框选。
Shift + 左键追加选择对象。不会清空已有选择。
Cmd/Ctrl + 左键切换对象的选中状态。已选中的对象会被移出选择，未选中的对象会被加入选择。
双击对象选择并聚焦对象。适合快速定位场景中的对象。
Shift + 拖拽框选时追加对象。用于扩大当前选择集。
Cmd/Ctrl + 拖拽框选时切换对象选中状态。用于从选择集中加入或移除一批对象。

选择

Scene View 和层级树都会驱动同一组选择状态。正常情况下，层级高亮、Scene View 选择目标和 Inspector 摘要应该指向同一对象。

入口 作用 说明
Scene View 点击选择被点击的 GameObject。对象需要可选中，并且项目 adapter 需要提供稳定 runtime binding。
层级树点击从 Hierarchy 中选择 GameObject。适合 Scene View 中对象重叠或难以点中时使用。
层级树 Shift / Cmd/Ctrl追加或切换选择。和 Scene View 多选语义一致。
右键层级节点打开 GameObject 操作菜单。可执行聚焦、重命名、创建子分组、分组选中对象、删除等操作。

文档操作

保存、撤销、重做、删除和复制都依赖项目 adapter 或接入宿主提供对应语义。快捷键触发后，如果项目拒绝操作，编辑器应该保持当前数据不被破坏。

快捷键 作用 说明
Cmd/Ctrl + S保存场景。保存 editor-scene，并由项目 adapter 编译或导出运行时 scene 数据。
Cmd/Ctrl + Z撤销。撤销上一步文档操作，前提是项目提供可撤销 patch。
Cmd/Ctrl + Shift + Z重做。恢复刚撤销的文档操作。
Ctrl + Y重做。兼容 Windows 风格重做快捷键。
Delete / Backspace删除当前选中的 GameObject。受 protected、locked、read-only 和项目删除规则限制。
Cmd/Ctrl + D复制当前选中对象。需要项目接入 Duplicate 规则；如果菜单或操作置灰，说明当前项目未启用。

层级

操作 作用 说明
右键节点打开节点上下文菜单。可聚焦、重命名、创建子分组、分组选中对象或删除。
F / Focus in Preview聚焦当前节点。节点需要可选中、未锁定且非 protected。
Rename开始重命名节点。read-only、locked 或 protected 节点不可重命名。
重命名时 Enter提交重命名。输入框失焦时也会提交。
重命名时 Esc取消本次重命名。保留原名称。
Add Empty Group创建空分组。在节点菜单中创建子分组；在空白区域菜单中创建顶层分组。
Group Selection把当前可移动的多选节点编入新分组。要求项目已连接分组 pipeline。
Delete删除选中节点。只删除允许删除的选中节点。

面板

入口 作用 说明
Assets 标签搜索项目资产，点击资产创建 GameObject。资产列表和创建规则由项目 adapter 提供。
History 标签查看来自 EditorSession 的文档操作历史。用于确认撤销/重做链路。
Inspector 输入框编辑选中对象的序列化属性。Transform 数值通常支持直接编辑。
多选 Inspector批量编辑多个选中 GameObject 的共享 Transform 数值。只应编辑所有选中对象共享且可持久化的字段。

限制与注意

  1. 快捷键需要编辑器或 Scene View 获得输入焦点；焦点在浏览器地址栏、外部宿主 UI 或文本输入框时，快捷键不会触发场景编辑。
  2. Q/W/E/R 在普通编辑状态下切换工具；右键飞行视角中，WASD/QE 用于移动相机。
  3. 删除、复制、资产创建、Inspector 字段写入和保存都可能被项目 adapter 拒绝。
  4. 层级菜单中的 Copy、Paste、Copy Transform、Paste Transform、Locked、Do Not Serialize 等占位项如果置灰，表示当前版本或当前项目尚未启用。
  5. Scene View 相机移动、平移、环绕和 dolly 只改变观察视角，不会写入场景数据。
  6. 当前版本记录默认快捷键；如果你的项目提供自定义绑定界面，以项目说明为准。

快捷键不生效

先看焦点：大部分快捷键失效都和输入焦点有关。先点击 Scene View 或编辑器空白处，再重试快捷键。
  1. 如果 Q/W/E/R 不切换工具，确认没有正在编辑 Inspector 输入框或层级重命名输入框。
  2. 如果 Cmd/Ctrl + S 没有保存，确认当前处于 Editor Mode，且项目没有处于 busy 状态。
  3. 如果 Delete 没有删除，检查选中节点是否 protected、locked、read-only，或者项目是否拒绝删除。
  4. 如果右键飞行视角无法移动，确认右键仍按住，并且 Scene View 处于活动状态。
  5. 如果复制、分组或资产创建没有效果，检查当前项目是否提供了对应 adapter 能力。
